# Data

Raw polysomnography data is not included in this repository due to size and privacy constraints.

## Data Structure
Each participant folder (AP01-AP05) contains:
- Flow.txt          — Nasal airflow signal (32 Hz)
- Thorac.txt        — Thoracic movement signal (32 Hz)
- SPO2.txt          — Blood oxygen signal (4 Hz)
- Flow Events.txt   — Annotated breathing events
- Sleep profile.txt — Sleep stage annotations
